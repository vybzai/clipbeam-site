Built for coding agents

Beam a file onto your remote box.
Get the path.

clipbeam is one static Go binary that pushes a file, screenshot, or clipboard from your laptop onto a remote box's disk — over your existing SSH or Tailscale — and prints the absolute path your agent can read.

Install clipbeam See the flow

laptop → box

# once, to pair your remote box
$ clipbeam setup user@box

# beam a screenshot onto the box
$ clipbeam send screenshot.png box
/home/user/.clipbeam/in/screenshot.png
 
# on the box — your agent reads it by path
$ claude "describe the image at $(clipbeam last)"

The flow

Three commands from laptop to agent

Universal Clipboard stops at the SSH boundary. A remote TUI can only ingest a real on-disk file by absolute path — so clipbeam writes the file and prints the path. That's the whole point.

01 · PAIR

Point at the box

One-time setup over your existing SSH. Host keys are verified; no new account, no daemon left running.

clipbeam setup user@box
02 · BEAM

Send the payload

A file, a screenshot, or your clipboard lands on the box's disk. clipbeam prints the absolute remote path.

clipbeam send screenshot.png box
03 · READ

The agent reads it

Claude Code, aider, or any tool on the box opens the file by path. clipbeam last echoes it back.

claude "...at $(clipbeam last)"
Designed for automation

A CLI agents can actually drive

Every verb speaks JSON, has a deterministic exit-code table, and describes itself. No screen-scraping human output. There's even a private agent-to-agent channel — an inbox drained by clipbeam recv.

  • --json on every command for clean machine parsing
  • Deterministic exit codes — agents branch on outcome, not text
  • clipbeam schema — self-describing surface, no guesswork
  • Agent-to-agent inbox drained by clipbeam recv
$ clipbeam send shot.png box --json

Quietly safe by default

Boring, predictable security

No new attack surface. clipbeam rides the tunnel you already trust and keeps secrets where macOS keeps them.

Push-only by design

Nothing leaves a machine unless you run a command. No polling, no background sync, no surprises.

SSH transport, verified

Rides your existing SSH tunnel with host keys verified — no new account. Tailscale is optional.

Secrets stay put

Tokens live in the macOS Keychain or a 0600 token file. Logs redact the token.

Private agent channel

An agent-to-agent inbox that never touches the human's clipboard or notifications — drained by clipbeam recv.

50 MB payload cap

A sane ceiling for screenshots, logs, and small assets. Big enough to be useful, small enough to stay snappy.

One binary, zero deps

A single static Go binary. Nothing to install on the box, no runtime, no daemon to babysit.
