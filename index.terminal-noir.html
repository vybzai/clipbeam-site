Three verbs close the gap between your eyes and the agent's disk

A coding agent on a headless box can't see your screen and can't read a file it doesn't have. Universal Clipboard dies at the SSH boundary. clipbeam closes that gap with one binary on each side.

1 · Beam

clipbeam send shot.png box pushes the bytes onto the box over your existing tunnel.

2 · Print the path

clipbeam writes the file and prints its absolute path — ready to drop into a prompt or a shell substitution.

3 · Agent reads it

claude "explain $(clipbeam last)" — the agent ingests a real on-disk file by its bare absolute path.

what's inside

Small surface, exactly the right verbs

Screenshots, files, clipboard — plus a private agent-to-agent channel that never touches the human's clipboard or notifications.

Screenshots & files

clipbeam shot grabs the screen; send beams any file onto the box's disk.

Clipboard, beamed

clipbeam push sends your clipboard to the box — no iCloud round-trip, no Universal Clipboard.

Agent-to-agent channel

A private inbox drained by clipbeam recv. Two agents hand off without touching the human's clipboard.

Push-only by design

Nothing leaves a machine unless you run a command. SSH host keys verified; no new account, no inbound port.

SSH or Tailscale

By default it rides your existing SSH tunnel. Prefer a tailnet? Point it at Tailscale instead.

Self-describing

clipbeam schema emits a machine-readable description of every verb — the agent's entry point.

built for agents

An agent can drive every verb without guessing

clipbeam doesn't just tolerate automation — it's designed for it. Deterministic, scriptable, introspectable.

  • →--json on every verb. Structured output an agent parses, not prose it scrapes.
  • →A deterministic exit-code table. Same failure → same code, every time.
  • →Self-describing via clipbeam schema. Read the contract first, then act.
  • →A private agent-to-agent inbox. recv drains it — invisible to the human.

trust boundary

Quiet by default. You hold every key.

clipbeam moves bytes you explicitly hand it, over a tunnel you already trust, to one box you already use.

  • Push-only. Nothing leaves a machine unless you act — it never polls or scrapes.
  • Rides your SSH. Host keys verified, no new account, no inbound port. Tailscale optional.
  • Secrets stay put. macOS Keychain or a 0600 token file. Logs redact the token.
  • 50 MB payload cap. Bounded transfers; large files fail loud, not silently.
